# oxymon
*Rust-based SpO2 Monitoring Project*

This project focuses on developing a SpO2 monitoring system using the MAX30105 sensor mounted on a SparkFun breakout board, interfaced with the STM32F429I-DISC1 discovery board.

## ⚠️ DISCLAIMER: NOT FOR MEDICAL USE

This project is a hobby/educational implementation of SpO2 monitoring and is NOT intended for medical, diagnostic, or clinical use. It has not been validated, tested, or approved for medical applications. The accuracy and reliability of SpO2 readings cannot be guaranteed.

Do NOT use this device to make medical decisions or diagnose any medical conditions. Always consult with qualified healthcare professionals for any health-related concerns.

## Introduction
The MAX30105 sensor is a versatile sensor designed for heart rate and SpO2 monitoring. This project uses Rust programming language for firmware development, ensuring safety and concurrency, along with real-time performance.

## Components
- **MAX30105 sensor**: Used for measuring heart rate and SpO2 levels.
- **SparkFun MAX30105 Breakout Board**: A ready-to-use board featuring the MAX30105 sensor.
- **STM32F429I-DISC1**: A powerful microcontroller development board from STMicroelectronics.
